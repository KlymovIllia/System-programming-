Основи Linux

Коротка інформація про ОС GNU/Linux:

Linux — це родина операційних систем, заснованих на ядрі Linux. Вони є вільними, відкритими та використовуються у всьому світі на серверах, комп'ютерах, мобільних пристроях та інших пристроях.

Популярні дистрибутиви Linux:

  • Ubuntu
  • Debian
  • Fedora
  • Arch Linux
  • Linux Mint

Основні команди Linux CLI:

  • pwd — Показати поточну директорію
  • echo — Вивести текст на екран
  • ls — Показати список файлів у директорії
  • cd — Переміститися до іншої директорії
  • touch — Створити новий файл
  • mkdir — Створити нову директорію
  • cp — Копіювати файли або директорії
  • mv — Перемістити або перейменувати файли
  • rm — Видалити файли або директорії
  • history — Показати історію виконаних команд

BASH-cкрипт для імітації авторизації:

#!/bin/bash
echo "Введіть ваше ім'я користувача:"
read username

if [ "$username" == "admin" ]; then
    echo "Авторизація успішна, $username!"
else
    echo "Авторизація не вдалася. Невірне ім'я користувача."
fi
